[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Init
Template for below stack
- Nestjs 
- graphql
- typeorm (if you want to use)
- nodejs v18.19

## Installation

개행문자 처리
```shell
git config --global core.autocrlf true
```

```bash
$ yarn install
```

## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## Test

```bash
# unit tests
$ yarn run test

# e2e tests
$ yarn run test:e2e

# test coverage
$ yarn run test:cov
```
